# Spatial Omics Research Digest

An automated daily archive of cutting-edge spatial-omics data-modeling research.

Once configured, GitHub Actions will:

1. Search recent primary research sources.
2. Select important new modeling papers.
3. Generate technical SVG summaries.
4. Commit a dated Markdown digest.
5. Open a GitHub Issue for notifications.

See [SETUP.md](SETUP.md) to connect the repository and add the required API key.
